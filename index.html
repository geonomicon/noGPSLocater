NO GPS Compass.

Android Location Detection(Lat,Lng) and Direction Compass which can be used when no GPS connection is available. The Application uses Mobile Country Codes and Mobile Network Codes to pin point the location of the user. Location obtained is usually of lower precision than GPS but higher precision than GPS is closed rooms where GPS signal is not enough to pinpoint user's location.

API Used

Location

Combain WiFi Based Positioning API is used to get the location in Lat, Lng from MNC, MCC and CID(Cell Id) of mobile network which we can get by using GsmCellLocation and TelephonyManager classes of Android.

Direction

For Direction identification, two types of compass are there, High Precision High Power consumption Magnetometer Compass and Low Precision Low Power Consumption Accelerometer Compass.

BaseMaps

Google Maps is used for Basemaps for plotting Location obtained from the Combain API

Database

SQLite Database for local storing of last location for tracking purposes. All location point stored are retrieved from the API and not from the GPS. Location is stored as Lat, Lng along with MNC, MCC, CellId and Magnetometer reading.
